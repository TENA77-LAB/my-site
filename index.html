Это исходный index.html для режима разработки (npm run dev). При открытии файла двойным щелчком React не подключается — поэтому вы видите эту подсказку.

Как смотреть сайт без сервера: дважды нажмите Открыть-сайт.bat — появится папка site и откроется готовый сайт. Дальше можно всегда открывать site\index.html или файл ОТКРОЙ-САЙТ.html в этой же папке.

Для правок в коде удобнее терминал: npm install → npm run dev → браузер http://localhost:5173.
